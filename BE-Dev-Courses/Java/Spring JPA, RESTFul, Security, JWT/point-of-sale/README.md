## 💻 About The Project

In the development of backend applications there are several aspects to be studied and learned, personally whenever I start the study of a new technology I like to use a course with a small project to have an overview of this technology.

That's why this project was done using the course [Practical Spring Boot JPA, RESTFul, Security, JWT][course-link]

And the certificate generated at the end of the course can be accessed at this [link][certificate]


## Technologies

This project was developed to study the following points:

- [Spring][spring]
- [Json Web Token][jsonwebtoken]
- [Kotlin][kotlin]
